# 你吃的那碗飯，根本不會讓你變胖

你是不是也這樣？

筷子夾了一口飯
停在半空中
又放回碗裡

你看了一眼旁邊同事的便當
她飯吃得乾乾淨淨
你的那口飯還在筷子上猶豫

「半碗就好」
「今天中午已經吃了麵包了」
「算了不要好了」

你把飯撥到碗的角落
假裝自己吃飽了

忍完以後呢？

晚上九點
你坐在沙發上
肚子開始叫

你知道不該吃
但腦袋裡一直轉的是鹹酥雞的味道

最後你走到冰箱前
冷光照在你臉上
你拆了一包巧克力餅乾
站在廚房邊吃邊罵自己

「又破功了」
「我怎麼這麼沒用」
「明天一定要少吃」

然後明天
一模一樣的循環再來一次

我自己以前也這樣

吃了一碗滷肉飯
明明很開心
放下筷子的瞬間就開始算

然後隔天少吃一餐「補回來」
結果到下午餓到去買波蘿麵包配奶茶

我花了很久才搞懂
問題根本不在那碗滷肉飯

我要很認真的告訴你

「問題從來不是那碗飯」
「問題是你吃完之後對自己說的那句話」

#你的罪惡感正在讓你變胖

這不是比喻
這是真的在你身體裡發生的事

你吃了那碗飯
心裡開始不安

「完了，今天又吃多了」

這個念頭本身
就是一個壓力訊號

你的身體收到壓力
會做一件事：
分泌[皮質醇](/concepts/cortisol)

皮質醇是壓力荷爾蒙
它一出來
會把你的血糖推高

血糖被推高
[胰島素](/concepts/insulin-resistance)就得出來處理

胰島素一出來
脂肪庫的門就關上了

所以光是「覺得自己吃錯了」這個念頭
就已經在鎖住你的脂肪了

然後你為了「補救」
隔天早餐不吃

空腹一整個上午
身體更確定了

「她在鬧飢荒」
「把所有東西存起來」

代謝降下來
脂肪鎖得更緊

到了中午
你餓到受不了
買了一個麵包配奶茶

空腹灌進去
[血糖](/concepts/blood-sugar-rollercoaster)瞬間衝上去
胰島素瘋狂出動

這一餐的脂肪儲存效率
比你正常吃飯高出好幾倍

然後晚上你又覺得罪惡
明天又少吃

循環就這樣越轉越緊

你看到了嗎

從頭到尾
讓你變胖的不是那碗飯

是你吃完之後的罪惡感
啟動了整條代謝災難鏈

罪惡感 → 壓力 → 皮質醇 → 血糖被推高
→ 胰島素鎖住脂肪 → 少吃補救
→ 身體以為飢荒 → [代謝下降](/concepts/metabolic-adaptation)
→ 餓到暴吃 → 胰島素瘋狂加班
→ 脂肪加倍儲存 → 更胖 → 更罪惡

這個循環的起點
不是食物
是你對食物說的那句話

#那如果不罪惡，那碗飯會怎樣？

完全不一樣

你正常吃一碗飯
血糖穩穩地上去

胰島素出來做一下事
把糖搬進肌肉當燃料
搞定了就下班

你的肌肉就像一塊海綿
平常有在走路、有在動
海綿是乾的
飯進去直接被吸走

根本輪不到變成脂肪

到了下一餐
你正常餓了
正常吃

血糖穩定
胰島素正常上下班
脂肪庫的門是開著的
身體有空間去燒庫存

同一碗飯
吃完安心的人 → 代謝正常運作
吃完罪惡的人 → 啟動整條災難鏈

差別不在那碗飯
在你吃完之後的心情

#那要怎麼打斷這個循環？

不是吃更少
不是更認真控制

是「停止懲罰自己」

你的身體不怕那碗飯
它怕的是你吃完之後的反應

當你吃了飯不罪惡
不隔天少吃「補回來」
不餓到下午再暴吃

血糖穩穩的
胰島素正常上下班
脂肪庫的門開著

循環就斷了

#「不怕澱粉」懶人包

「第一：吃了就吃了，下一餐正常吃」

不要「補回來」
你一補
就啟動飢荒模式
反而更慘

每一餐都是新的開始
上一餐吃了什麼不重要
下一餐好好吃就好

「第二：讓飯慢慢進去，不要用衝的」

吃便當的時候
先吃燙青菜
再吃滷雞腿或烤魚
最後再扒飯

外面吃麵的話
先把湯裡的肉和菜撈起來吃
麵最後吃

同樣的東西
[這樣吃](/articles/eating-order-blood-sugar)血糖穩很多
胰島素不用瘋狂出動

「第三：吃到飽，不要吃到撐」

吃飽跟吃撐是兩件事

一份燙青菜、一隻雞腿、半碗飯
你吃得飽
身體收到「營養夠了」的訊號

一份炒飯配一杯珍奶
你吃得撐
但身體覺得營養是空的

差別在你給身體的是「訊號」還是「空的熱量」

當你不再為每一口飯懲罰自己
你的身體反而會開始站在你這邊
